Open Source

Some open source software I created and maintain:
Zine
A pretty fast static site generator written in Zig.
SuperHTML
HTML validator, formatter, LSP, and templating language library.
SuperHTML is the first language server to implement the full HTML spec!
Ziggy
A data serialization language for expressing clear API messages, config files, etc.
Awebo
A self-hostable Discord alternative written from scratch (currently pre-alpha).
